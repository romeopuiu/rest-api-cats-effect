# REST API with Cats Effect

A purely functional CRUD REST API built with Scala 3 and the Typelevel ecosystem.  
---
## 🛠 Technologies Used

- **Scala 3** – Main programming language
- **Cats Effect** – Functional runtime and effect system
- **http4s (Ember)** – Functional HTTP server
- **Doobie** – Type-safe database access
- **PostgreSQL** – Relational database
- **Circe** – JSON encoding/decoding
- **FS2** – Functional streaming
- **EitherT** – Explicit domain error handling
- **Docker** – Containerization
- **Java 21** – JVM runtime
- **sbt** – Build tool

---
